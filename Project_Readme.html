Interview ASP.Net Project - Hieu nguyen & Long Nguyen

Project Progress Overview

Brief Introduction

  •         Interview is an ASP.Net MVC project in development with the purpose of helping computer science graduates or entry level developers finding resources of interview questions. The website is not a place intended to study for interview questions but rather it serves as a forum for the users to share experiences with one another. This include posting any questions they encountered from previous interviews or help answering any question their peers might have. The whole purpose is to practice the technical techniques and problem solving skills in different topics such as algorithms, data strutures, software development, operating system and more.

Current Project Scope

  • At the current stage, most of the application features are finished. What our group will have to do now is to refactor some code, perform some layout editing and lastly is to perform all the necessary testings.

Implemented Features

  1. Anonymous users can view interview questions and comments. (Does not require login)
  2. User can register and login.
  3. User can post interview questions and perform CRUD operations on his/her own posts.
  4. User can comment on interview questions and perform CRUD operations on his/her own comments.
  5. User has their own profile page which can be view by other users.
  6. User can attach tags to their questions/posts.
  7. User can upvote or downvote the questions and comments. They can also cancel their vote anytime.
  8. User can flag/report negative questions and comments to the admin. They can also cancel their unflag or cancel the report anytime
  9. Admin can view all users, all posts and reported posts in the admin panel.
  10. Admin can delete any post or grant admin access to other users.
  11. Other implemented features include repository pattern, and dependency injection to keep database logic separate from the controllers.
